
# Motion
![Motion](img/Motion.jpg)

Motion lets you play the motion set for the your avatar in HEOWorldSetting. Specify the name in the Motion Name.

|  Label |  function  |
| ----   | ---- |
| name | Set the variable name as you like. |
| nvalue | Assign integer value. |
